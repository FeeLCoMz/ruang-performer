# PerformerHub

Aplikasi web untuk manajemen chord, setlist, dan performa musik.

## Struktur Folder
- `src/` : Kode frontend (React)
- `api/` : API backend
- `db/` : Skema dan migrasi database
- `public/` : Aset statis

## Instalasi
```bash
npm install
```

## Menjalankan
```bash
npm run dev
```

## Testing
```bash
npm test
```

## Environment
Lihat `.env.example` untuk variabel yang dibutuhkan.
